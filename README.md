## 🙌 소개
- 브릿지는 개발자, 디자이너, 기획자들을 위한 사이드 프로젝트 모집 플랫폼입니다.
- 저희는 실제로 프로젝트를 모집하거나 구하며 느꼈던 불편함들을 해소하고 싶었기에 이 프로젝트를 시작하게 되었습니다.
- 사용 기술: UIKit, RxSwift, FlexLayout, Starscream, SPM, SwiftLint
